# User Manual

## Citizen Features

Citizens use the portal to register civic complaints and track their progress.

## Registration

1. Open the application.
2. Click **Register**.
3. Enter full name, email, phone number, password, security question, and answer.
4. Submit the form.
5. Login using the registered email and password.

## Login

1. Open the login page.
2. Enter email and password.
3. Optional: select **Remember Me**.
4. Click **Login**.

## Submit Complaint

1. Go to **Submit Complaint**.
2. Enter title and description.
3. Select category, priority, and ward.
4. Enter location.
5. Optionally upload a before image.
6. Submit the complaint.

The system automatically generates a complaint number in this format:

```text
LCP-YYYY-XXXXXX
```

## Complaint History

Citizens can view all submitted complaints with:

- Complaint number
- Title
- Status
- Priority
- Submitted date

Citizens can search by complaint number, status, and category.

## Complaint Details

The detail page displays:

- Complaint information
- Before image
- After image, if uploaded by officer
- Resolution notes
- Timeline of updates

## Notifications

Citizens receive notifications for:

- Complaint submission
- Assignment
- Status updates
- Resolution
- Closure

Citizens can mark individual notifications or all notifications as read.

## Profile

Citizens can update:

- Full name
- Phone number
- Password

## Forgot Password

Password reset is handled through security question verification.
